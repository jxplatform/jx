# Getting Started with JSONsx

JSONsx lets you build entire websites using JSON. No complex build tools, no framework lock-in — just declarative documents that compile to static HTML.

## Why JSONsx?

- **Declarative** — describe what you want, not how to build it
- **Zero runtime** — static HTML output by default
- **Content collections** — Markdown, JSON, and CSV content with schema validation
- **Layouts and slots** — reuse page shells across your site

Start by creating a `site.json` in your project root.
